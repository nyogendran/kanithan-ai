How VAD solves student pauses mid-sentence:
The core problem: a Grade 7 student says "72 உம் 108 உம்..." then pauses 1.5s while thinking "...ஆகிய எண்களின் போ.கா.பெ. காண்க". Without intelligent VAD, a simple silence timeout cuts off mid-sentence.
Speech detected→ Pause ≤ 2.5s→ VAD: still active→ Speech resumes→ Pause > 2.5s + math complete→ Finalize utterance
Silero VAD (30ms chunks, 92kB ONNX model, runs on-device) gives speech probability 0.0–1.0 per frame.
Adaptive timeout: 1.5s for short answers, 3.0s for problem-stating, 4.5s when numbers detected mid-utterance.
Math pause guard: if partial transcript contains digit words (இரண்டு, மூன்று) or operators, extend timeout.
