Teaching Explanation (Tamil)
18, 180, 162 ஆகிய எண்களின் பொ.கா.பெ.யைக் காரணி மரம் முறை மூலம் காண்போம்.

முதலில், ஒவ்வொரு எண்ணையும் முதன்மைக் காரணிகளின் பெருக்கமாக எழுதுவோம்:

**1. 18 இன் முதன்மைக் காரணிகள்:**

```
      18
     /  \
    2    9
        / \
       3   3
```
ஆகவே, 18 = 2 × 3 × 3 = 2¹ × 3²

**2. 180 இன் முதன்மைக் காரணிகள்:**

```
       180
      /   \
     2     90
          /  \
         2    45
             /  \
            3    15
                /  \
               3    5
```
ஆகவே, 180 = 2 × 2 × 3 × 3 × 5 = 2² × 3² × 5¹

**3. 162 இன் முதன்மைக் காரணிகள்:**

```
       162
      /   \
     2     81
          /  \
         3    27
             /  \
            3    9
                / \
               3   3
```
ஆகவே, 162 = 2 × 3 × 3 × 3 × 3 = 2¹ × 3⁴

இப்போது, மூன்று எண்களுக்கும் பொதுவான முதன்மைக் காரணிகளைப் பார்ப்போம். ஒவ்வொரு முதன்மைக் காரணியிலும் மிகக் குறைந்த அடுக்கை எடுக்க வேண்டும்:

*   **2:** 18 இல் 2¹, 180 இல் 2², 162 இல் 2¹. இவற்றில் மிகக் குறைந்த அடுக்கு 2¹.
*   **3:** 18 இல் 3², 180 இல் 3², 162 இல் 3⁴. இவற்றில் மிகக் குறைந்த அடுக்கு 3².
*   **5:** 180 இல் 5¹ உள்ளது, ஆனால் 18 மற்றும் 162 இல் 5 இல்லை. எனவே, 5 பொதுவான காரணி அல்ல.

பொ.கா.பெ. = பொதுவான முதன்மைக் காரணிகளின் மிகக் குறைந்த அடுக்குகளின் பெருக்கம்
பொ.கா.பெ. = 2¹ × 3²
பொ.கா.பெ. = 2 × (3 × 3)
பொ.கா.பெ. = 2 × 9
பொ.கா.பெ. = 18

ஆகவே, 18, 180, 162 ஆகிய எண்களின் பொ.கா.பெ. 18 ஆகும்.

மாணவரே, காரணி மரத்தில் ஒவ்வொரு எண்ணையும் முதன்மை எண்களாகப் பிரிக்கும்போது, கடைசி வரிசையில் உள்ள எண்கள் அனைத்தும் முதன்மை எண்களா என்பதை எப்படிச் சரிபார்ப்பீர்கள்?
முதன்மைக் காரணி என்றால் என்ன, ஒரு எண் 15 ஆல் வகுபடுமா என்பதை எப்படித் தெரிந்துகொள்வது போன்ற கூடுதல் கேள்விகள் இருந்தால் கேட்கலாம்.
